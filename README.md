**MÔ TẢ DỰ ÁN**

TFitnessApp là một ứng dụng desktop được xây dựng bằng WPF (.NET) và ngôn ngữ C# nhằm cung cấp một giải pháp quản lý cho các trung tâm gym quy mô vừa và nhỏ. Ứng dụng tập trung vào việc số hóa quy trình vận hành, loại bỏ sự phụ thuộc vào sổ sách hay Excel , từ đó nâng cao hiệu quả quản lý hội viên, hợp đồng, giao dịch và chất lượng dịch vụ.

---
**MỤC TIÊU ĐẠT ĐƯỢC** 

Hỗ trợ nghiệp vụ toàn diện: Xây dựng một ứng dụng Desktop ổn định, hỗ trợ đầy đủ 11 nhóm chức năng nghiệp vụ cốt lõi (Quản lý Học viên, Gói tập, Hợp đồng, Lịch tập, Giao dịch, Điểm danh, v.v.).

Thiết kế Hiện đại: Phát triển giao diện người dùng (UI/UX) trực quan, hiện đại bằng WPF, với tông màu chủ đạo Đỏ/Trắng , phân quyền rõ ràng giữa Quản trị viên và Nhân viên.

Thiết kế cơ sở dữ liệu SQLite với 13 thực thể quan hệ chặt chẽ, đảm bảo tính toàn vẹn và hiệu quả truy xuất.

Trực quan hóa Dữ liệu: Tích hợp thư viện LiveChart để tạo các biểu đồ trực quan (Dashboard, Báo cáo Doanh thu) hỗ trợ ra quyết định kinh doanh kịp thời.

---
**CÔNG NGHỆ & KỸ THUẬT ÁP DỤNG**

Ứng dụng desktop Quản lý Trung tâm Gym TFitness được xây dựng trên nền tảng .NET Framework để đảm bảo hiệu suất cao và ổn định, với C# là ngôn ngữ lập trình chính chịu trách nhiệm xử lý toàn bộ logic nghiệp vụ (backend) và tương tác với cơ sở dữ liệu. Về mặt giao diện, dự án sử dụng WPF (Windows Presentation Foundation) để thiết kế UI/UX phong phú, hỗ trợ mạnh mẽ các kỹ thuật Databinding và Animation nhằm mang lại trải nghiệm mượt mà và trực quan cho người dùng.

Hệ thống quản lý dữ liệu sử dụng SQLite làm hệ quản trị cơ sở dữ liệu cục bộ, lý tưởng cho ứng dụng desktop, đảm bảo khả năng truy xuất nhanh chóng. Về kỹ thuật cơ sở dữ liệu, nhóm đã sử dụng LINQ và ADO.NET (SQLite) để thực hiện các truy vấn và thao tác dữ liệu phức tạp, đồng thời áp dụng các kỹ thuật kiểm tra ràng buộc để đảm bảo tính toàn vẹn và định dạng dữ liệu đầu vào (Email, SĐT, Giá trị số). Cuối cùng, để hỗ trợ trực quan hóa dữ liệu, thư viện LiveChart đã được tích hợp để tạo các biểu đồ (đường, cột, tròn) cho module Báo cáo và trang Tổng quan.

**HƯỚNG DẪN TRIỂN KHAI & ĐĂNG NHẬP**

Yêu cầu Hệ thống:

+ Hệ điều hành: Windows

+ Môi trường: .NET Framework (hoặc .NET Core tương thích)

Triển khai Nhanh: 

+ Download file ZIP về máy của bạn

+ Mở thư mục dự án và tìm file TFitnessApp.sln.

+ Mở file TFitnessApp.sln bằng Visual Studio.

+ Xây dựng (Build) và chạy (Run) dự án.

---
**HƯỚNG PHÁT TRIỂN TƯƠNG LAI**

Thống kê và phân tích dự báo: Phát triển module thống kê và dự đoán về học viên (tần suất luyện tập, khả năng gia hạn hợp đồng) nhằm đạt hiệu quả quản lý tốt hơn.

Tích hợp API Dinh dưỡng: Tích hợp API Spoonacular để đề xuất thực đơn dinh dưỡng hàng tuần theo mục tiêu sức khỏe cá nhân của học viên.

Mở rộng Cloud & API: Chuyển đổi CSDL từ SQLite sang SQL Server hoặc PostgreSQL, xây dựng API trung gian để hỗ trợ đồng bộ dữ liệu và mở rộng ứng dụng lên nền tảng web/mobile.
